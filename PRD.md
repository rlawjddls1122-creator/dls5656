# PRD — 오늘의 작업 BGM (Today's Work BGM)

## 서비스명
오늘의 작업 BGM (Today's Work BGM)

## 한 줄 설명
지금 내 기분과 작업 종류에 맞는 플레이리스트를 추천해주는 큐레이션 도구

## 핵심 기능 3가지
1. 기분 태그 + 작업 태그 기반 플레이리스트 필터링
2. 유튜브 / 스포티파이 외부 링크 플레이리스트 카드
3. 랜덤 추천 "오늘의 픽"

## 대상 유저
작업할 때 음악을 틀어놓는 디자이너, 개발자, 글쓰기 작업자

## 사용 기술
Next.js + Tailwind CSS

## 디자인 톤
다크 모드, 미니멀, 차분함, 몰입감
